# MVP Scope

## In Scope

- Receive healthcare operational events through an API.
- Analyze operational events using an LLM.
- Generate structured AI responses.
- Answer healthcare operational queries using internal SOPs and knowledge.
- Explain operational comments in simple language.
- Generate daily operational summaries.
- Recommend actions based on operational events.
- Prioritize critical operational cases.
- Trigger automation through n8n (where applicable).

---

## Out of Scope

- Patient-facing application
- Appointment booking
- Hospital Management System (HMS)
- CRM replacement
- Billing and invoicing
- User authentication and authorization (initial MVP)
- Advanced analytics dashboards
- Mobile application
- Multi-tenant support

---

## Future Enhancements

- MCP Integration
- Role-Based Access Control (RBAC)
- Multi-hospital support
- Voice Assistant
- Predictive operational insights
- Advanced process mining
